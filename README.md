# 合乎周礼

> 灵感与文案来源：[合乎周礼](https://hehuzhouli.com/) · 原作者 [NFS360](https://space.bilibili.com/455164654)

将现代中文改写为「周礼白话翻译腔」——一本正经、略显荒唐，像中学课文白话译文那样把小事说圆。

## 用法

```
周礼 <文本>
周礼 [篇幅] [辞气] <文本>
```

### 篇幅

| 关键词 | 字数 |
|--------|------|
| 小礼 | 70-130 |
| 成礼（默认） | 150-260 |
| 大礼 | 280-450 |

### 辞气

| 关键词 | 风格 |
|--------|------|
| 温言 | 体谅劝说 |
| 辩经 | 论证+反问 |
| 圆场 | 另立名分 |
| 痛心 | 郑重谴责 |

### 其他触发方式

- `@机器人 周礼 <文本>` — 群聊中 @机器人
- 引用一条消息 + `周礼` — 改写被引用的消息
- `周礼 help` / `周礼 帮助` — 查看帮助

## 配置

通过 WebUI 插件管理面板可配置：

- `command_prefix` — 触发前缀（默认：周礼）
- `default_length` — 默认篇幅（默认：成礼）
- `default_tone` — 默认辞气（默认：自动）
- `cooldown` — 用户冷却时间（秒，默认：5）
- `ignore_slash` — 是否忽略 `/` `#` 前缀（默认：true）
- `system_prompt` — 自定义改写提示词（留空使用内置）
